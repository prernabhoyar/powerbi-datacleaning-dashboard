# Data Cleaning & Reporting Automation

## Overview

This project demonstrates an automated data cleaning and reporting workflow using Power BI. The objective is to improve data quality by handling duplicates, cleaning textual data, validating data types, and generating interactive reports for business analysis.

## Objectives

* Automate data cleaning processes.
* Improve data consistency and accuracy.
* Generate visual reports and dashboards.
* Analyze sales and profit performance.
* Support data-driven decision-making.

## Tools Used

* Power BI
* Power Query

## Dataset

The project uses the Sample Superstore dataset containing:

* Order Details
* Customer Information
* Product Information
* Sales Data
* Profit Data
* Regional Information

## Data Cleaning Steps

1. Imported dataset into Power BI.
2. Removed duplicate records.
3. Trimmed unnecessary spaces from text fields.
4. Cleaned textual data for consistency.
5. Verified and corrected data types.
6. Prepared the dataset for reporting and visualization.

## Dashboard Features

### KPI Cards

* Total Sales
* Total Profit

### Visualizations

* Sales by Category (Bar Chart)
* Sales by Region (Pie Chart)
* Sales Trend by Order Date (Line Chart)

## Key Insights

* Total Sales reached approximately 2.30M.
* Total Profit reached approximately 286.40K.
* Technology category generated the highest sales.
* West region contributed the largest share of sales.
* Sales trends provide insights into business performance over time.

## Expected Outcomes

* Understanding of data preprocessing techniques.
* Improved data quality through cleaning automation.
* Automated reporting and visualization.
* Faster and more efficient business analysis.
